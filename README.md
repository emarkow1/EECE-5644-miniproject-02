# EECE-5644-miniproject-02

This project revolves around making decisions about potential customer's insurance charges using various machine learning models. Models of different kinds were created and evaluated against one another.

## Downloading the Dataset

The dataset [insurance-2.csv](insurance-2.csv) is already present in the repository, however, can be downloaded from [kaggle.com](https://www.kaggle.com/datasets/mirichoi0218/insurance)

## How to Run

All the necessary code is present in [miniproject02.ipynb](miniproject02.ipynb). First install the dependencies using 
```
pip install -r requirements.txt
```
